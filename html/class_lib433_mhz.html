Detailed Description

Gestion d'un émetteur/récepteur 433Mhz.

  • Documentation du Récepteur RXB-12 0
  • HowTo 1
  • HowTo 2

La librairie à utiliser est la librairie RadioHead
La documentation sur la librairie est disponible ICI

Attention
Il est nécessaire de modifier le fichier ** C:\Users\<username>\Documents\Arduino\Libraries\RadioHead\RH_ASK.cpp** et décommenter la ligne
#define RH_ASK_ARDUINO_USE_TIMER2
pour éviter les incompatibilité avec la librairie 'servo'

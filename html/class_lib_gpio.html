◆ configure()

bool LibGpio::configure ( unsigned char gpioIdx,
t_gpioMode gpioMode,
unsigned short defaultValue = 0 )

Permet de configurer le mode associé à un GPIO. Les modes disponibles sont les suivants :

  • PAMI_GPIO_UNUSED pour un GPIO non utilisé
  • PAMI_GPIO_INPUT pour un GPIO utilisé en entrée
  • PAMI_GPIO_OUTPUT pour un GPIO utilisé en sortie
  • PAMI_GPIO_PWM pour un GPIO utilisé en mode PWM
    Returns
    true ou false si le mode est inconnu ou si le GPIO n'existe pas
Parameters
gpioIdxNuméro d'index du GPIO à configurer (à partir de 0)
gpioModeType de configuration du pin
defaultValueValeur par défaut à appliquer

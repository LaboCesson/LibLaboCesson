Detailed Description

Description du role de la librairie LibMoteurOmni.

Cette librairie permet de piloter 4 roues Omnidirectionnelle

La fonction avance permet de controler le mouvement selon une des directions définies par t_omni_direction

Il est possible de controler indépendamment chacune des roues avec les fonctions suivantes

  • avantGauche
  • avantDroit
  • arriereGauche
  • arriereDroit

Detailed Description

Description de la librairie de gestion de plusieurs détecteurs de couleur TCS3472.

L'adresse I2C du composant TCS3472 est égal à 0x29 et ne peut être modifié

On utilise donc un multiplexeur I2C pour gérer plusieurs composants

Librairies requises : Adafruit_TCS34725, LibTcs3472

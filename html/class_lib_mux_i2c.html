Detailed Description

Gestion du multiplexeur I2C TCA9548A.

Le multiplexeur I2C TCA9548A permet de gérer jusqu'à 8 lignes I2C

Cela permet de gérer simultanément plusieurs composants I2C possédant la même adresse

La classe LibMuxI2c permet de sélectionner une des 8 voies I2C avant de pouvoir dialoguer avec un des composants présents sur la ligne sélectionnée

A l'initialisation, la ligne 0 est sélectionnée

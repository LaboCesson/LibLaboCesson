Detailed Description

Librairie de gestion du PAMI de l'année 2025.

Librairies de gestion des PAMI de l'année 2026.

Librairies requises :

MPU6050 : https://github.com/electroniccats/mpu6050

Servo : https://docs.arduino.cc/libraries/servo/

RadioHead : https://www.airspayce.com/mikem/arduino/RadioHead

Wire

Attention
En cas d'utilisation de la librairie Lib433Mhz, le timer 2 est utilisé à la fois pour gestion du 433Mhz et la génération d'un signal PWM sur la broche D11 qui est utilisé pour le pilotage du moteur gauche.
Dans le cas d'un PAMI avec un seul moteur, il faut utiliser le moteur droit (ENb)
Dans le cas d'un moteur avec deux moteurs, il faut utiliser valider le mode setPwmMode(false)
Dans ce dernier cas, il n'y a plus que les vitesses suivantes qui sont disponibles 0,20%,40%,60%,80%,100%

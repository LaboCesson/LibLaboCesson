Detailed Description

Description de la librairie de gestion d'un détecteur de couleur TCS3472.

Le composant TCS3472 permet de détecter les couleurs

Il permet de retourner les valeurs RGB et la temperature de la couleur

Librairies requises : Adafruit_TCS34725

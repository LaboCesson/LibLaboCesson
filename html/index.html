Documentation de la librairie LibLaboCesson

La librairie contient un ensemble de classes C++ permettant de gérer les différents éléments utilisés sur les robots développés au sein du LaboCesson

Les classes ont été compilées pour les famillles suivantes de microcontroleurs :

  • Arduino
  • ESP32

Les librairies ont été testées avec l'IDE Arduino . La page Installer l'IDE Arduino pour l'ESP32 décrit comment configurer l'IDE Arduino pour compiler pour un ESP32

La page Utilisation de la librairie LibLaboCesson donne plus d'informations sur la librairie et sa mise en oeuvre

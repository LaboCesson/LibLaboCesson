Documentation de la librairie LibLaboCesson

La librairie contient un ensemble de classes permettant de gérer les différents éléments utilisés sur les robots développés au sein du LaboCesson

Les classes ont été compilées pour les famillles suivantes de microcontroleurs :

  • Arduino
  • ESP32

Les librairies ont été testées avec l'IDE Arduino . La page Installer l'IDE Arduino pour l'ESP32 décrit comment configurer l'IDE Arduino pour compiler pour un ESP32

Les classes sont de deux types :

  • Des classes basiques permettant de gérer un composant (par exemple la classe LibAff1637 pour gérer un afficheur 7 Segments)
  • Des classes robot permettant de gérer l'ensemble des composants d'un robot (par exemple LibPami2026Basic gérant un PAMI basique du concours 2026)

Les classes basiques sont génériques et peuvent être utilisées dans un contexte quelconque. Par exemple la classe LibAff1637 permet de connecter l'afficheur sur n'importe quel port. Les classes robots utilisent les classes basiques en définissant à quel port l'afficheur est connecté

L'afficheur sera alors utilisable en utilisant par exemple un appel du type _pami2026basique.afficheur.displayValue(10)

Les classes **robots* gérent l'ensemble des éléments d'un robot, mais ils ne sont pas les logiciels finaux des robots. Les différents éléments accessibles par une classe de type robot seront utilisées pour par exemple :

  • Attendre un ordre de départ (par exemple avec la librairie LibAff1637)
  • Faire avancer le robot (par exemple avec la librairie LibMoteur)
  • S'assurer que le robot va droit (par exemple avec la librairie LibGyroscope)
